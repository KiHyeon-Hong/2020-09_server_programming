엘리먼트에는 블록 레벨 엘리먼트(다른 줄에 표현되는 엘리먼트)와 인라인 엘리먼트(같은 줄에 표현)가 있음

태그 속성 사용 규칙
소걱은 반드시 시작 태드에 명시
속성은 name="value"와 같이 속성 이름과 속성 값의 쌍으로 표현
속성 값은 작은따음표나 큰따음표 안에 표현
HTML4 명세에 속성 이름 및 속성 값으로 소문자를 사용하 것 권장

border-collapse:collapse - Table의 테두리 선과 각 셀의 테두리 선이 겹쳐 표현
border-collapse:separate - Table의 테두리 선과 각 셀의 테두리 선이 분리되어 표현

단일 파일 페이지 모델(Single-file page model)
업무 로직을 담고 있는 코드 부분과 시각적인 부분을 담당하는 HTML 부분이 하나의 파일(aspx 파일)에 위치한 모델

이벤트 기반 프로그래밍 - 다양하게 발생하는 이벤트들을 중심으로 프로그래밍하는 방식
